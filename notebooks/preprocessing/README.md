# Preprocessing Notebooks

This folder contains the notebooks used to construct the model-ready datasets from the original fire detection, meteorological, and land cover data.

The preprocessing workflow includes:
- Cleaning and preprocessing VIIRS active fire detection data and meteorological data
- Aggregating and validating fire detections to daily location-day observations
- Merging fire detections with meteorological and land cover data
- Exploratory data anlysis and fire radiative power (FRP) threshold selection
- Finalization of model-ready datasets used for classification

These notebooks correspond to the preprocessing and dataset construction steps described in sections 5–9 of the project report.
