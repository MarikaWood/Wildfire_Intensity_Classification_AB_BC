# Data

This folder documents the datasets used in this project. Dues to file size limitations, the full datasets are not included in this repository.

## Fire Detection Data

VIIRS S-NPP 375 m Active Fire Product data were downloaded from NASA FIRMS via Earthdata.

Source: 
https://www.earthdata.nasa.gov/

## Meteorological Data

Meteorological station observations were obtained from Environment and Climate Change Canada (ECCC) through the MSC Datamart.

Source:
https://www.canada.ca/en/environment-climate-change/services/weather-general-tools-resources/weather-tools-specialized-data.html

## Land Cover Data

Land cover information was obtained from the MODIS MCD12Q1 Land Cover Type product using the IGBP land cover classification system.

Source:
https://doi.org/10.5067/MODIS/MCD12Q1.061

## Dataset Reproducibility

The notebooks in 'notebooks/preprocessing/' contain all steps required to reconstruct the processed datasets used for modeling.
